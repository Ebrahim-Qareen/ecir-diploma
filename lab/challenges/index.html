Instructor · SPL solutions 
# Q1 brute-force source
index=endpoint_logs event_type=authentication result=failed | stats count by src_ip | sort - count
# Q2 first success from attacker IP
index=endpoint_logs event_type=authentication result=success src_ip="45.9.148.37" | sort + timestamp | table timestamp user
# Q3 download tool + C2
index=endpoint_logs event_type=process (command="*wget*" OR command="*curl*") | table timestamp user command
# Q4 + Q7 dropper name + hash
index=endpoint_logs event_type=file | table timestamp process file_name file_hash
# Q5 miner beacon (network, not command line)
index=endpoint_logs event_type=network dest_port=8080 | stats count by dest_ip dest_port
# Q6 persistence
index=endpoint_logs (process=useradd OR file_name="mysql-backup" OR command="*cron*") | table timestamp user process command file_name

Instructor · SPL solutions 
# Q1 dir brute-force source
index=web_logs status=404 | stats count by src_ip | sort - count
# Q2 tool (filter to the 404 flood first)
index=web_logs src_ip="193.106.191.22" status=404 | stats count by user_agent | sort - count
# Q3 login brute force + success (401 fail, 302 success)
index=web_logs uri="/admin/login" | stats count by status
# Q4 SQLi payloads
index=web_logs uri="*UNION*" OR uri="*'%20OR%20'1'='1*" | table timestamp uri status
# Q5 path traversal outcomes
index=web_logs uri="/download*" | table timestamp uri status
# Q6 web shell + commands
index=web_logs uri="*shell.php*" | sort + timestamp | table timestamp uri status
# Q7 secondary payload (URL-decode cmd)
index=web_logs uri="*shell.php*wget*" | table timestamp uri

# Q1 parent of powershell
index=sysmon EventID=1 Image="*powershell.exe" | table _time ParentImage Image CommandLine
# Q2 + Q3 LOLBin and the download URL

index=sysmon EventID=1 | table _time ParentImage Image CommandLine | sort _time

Instructor · SPL solutions 
# Q1 new user
index=wineventlog EventID=4720 | table _time SubjectUserName TargetUserName
# Q2 added to group
index=wineventlog EventID=4732 | table _time TargetUserName GroupName
# Q3 + Q4 malicious service (filter out SYSTEM's benign installs)
index=wineventlog EventID=7045 SubjectUserName="j.hassan" | table _time ServiceName ServiceFileName
# Q5 scheduled task
index=wineventlog EventID=4698 | table _time TaskName
# Q6 run-key persistence
index=wineventlog EventID=13 | table _time RegistryKey RegistryValue
# Q7 log cleared
index=wineventlog EventID=1102 | table _time SubjectUserName
How to run this in class

Give students the dataset + questions only. They ingest, investigate, and answer. Reveal the SPL and the answer feedback for grading. The Session 2 set (VPN · SSH · Web) covers SIEM basics; the Session 3 set (Linux auditd · Windows Sysmon · persistence) covers telemetry & SPL. Harder DFIR challenges arrive in Sessions 9–10.
