How to Read the Labels

Required vs Optional

Required labs are part of the homework and expected before the next session. Optional labs are for anyone who wants to go deeper.

In-class vs Homework

Some rooms are demonstrated live in class; most are done as homework so class time stays on the lab you build yourself.

INE stays primary

These labs reinforce the INE material — they never replace it. Where wording differs, INE's version is the one the eCIR exam tests.
